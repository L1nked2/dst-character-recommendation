# 질문지

추천 사이트에서 사용할 질문 목록입니다. 이 문서는 질문 문구와 선택지를 검토하고 수정하기 위한 작업용 문서입니다.

## 표기 규칙

- `weights`: 현재 구현체와의 비교를 위해 남겨둔 기존 성향 팩터입니다. 신규 구현은 [질문-팩터 매핑 v2](question-factor-map-v2.md)의 4축 벡터를 사용합니다.
- `UI 선택지`: 실제 질문 화면에 표시할 짧은 답변입니다.
- `문서용 설명`: 기획 검토용 설명입니다. 실제 질문 화면에는 표시하지 않습니다.
- 모든 질문은 4지선다로 구성합니다.
- Wonkey는 16가지 정규 유형에 넣지 않고 `WNKY` 와일드카드 유형으로 별도 처리합니다.

## 구현 메모

신규 추천 로직에서는 위 `weights`를 그대로 쓰지 않습니다. 질문은 다음 4축으로 변환합니다.

- `O/B`: Outfield / Base
- `D/S`: Direct / Setup
- `S/T`: Self / Team
- `E/H`: Easy / Hard

Wonkey는 16가지 정규 유형에 넣지 않고 `WNKY` 와일드카드 유형으로 별도 처리합니다.
